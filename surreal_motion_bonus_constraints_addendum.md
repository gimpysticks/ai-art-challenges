# Surreal Motion Bonus Constraints Addendum

## Motion & Reel Bonus Objectives

When applicable, prompts should support:

- Surreal motion design
- Cinematic short-loop animation concepts
- Storyboard-friendly scene sequencing
- Reel-ready visual pacing
- Dreamlike environmental movement
- Subtle uncanny looping behavior

## BONUS POINTS FOR

- Video
- Storyboards
- Short animation loops
- Reels with surreal motion

## Motion Design Guidance

Incorporate visual elements that imply movement even in still imagery:

- Floating or orbiting objects
- Repeating environmental patterns
- Glitch-like spatial shifts
- Looping atmospheric effects
- Slow drifting particles
- Breathing architecture
- Impossible physics
- Memory distortion
- Elastic environmental motion
- Temporal repetition
- Dream transitions
- Light flicker variations
- Soft camera drift
- Endless hallways, trains, escalators, corridors, or moving landscapes

## Reel / Animation Suggestions

When generating concepts for Reels or short-form video:

- Favor emotionally readable focal points
- Keep motion simple and hypnotic
- Use looping visual logic
- Emphasize atmospheric transitions over action
- Build scenes around 3–5 second seamless loops
- Suggest surreal environmental behaviors rather than chaotic motion
- Favor cinematic pacing over rapid cuts

## Visual Tone

Preferred visual tone for motion-oriented prompts:

- Dreamcore
- Soft surrealism
- Nostalgic uncanny imagery
- Emotional liminal spaces
- Gentle anime-inspired realism
- Candy-colored twilight palettes
- Glossy reflections
- Ambient cinematic lighting
- Emotional memory aesthetics

## Prompt Integration Rule

When appropriate, prompts may include subtle references to:

- looping motion
- animation-ready composition
- storyboard framing
- surreal environmental transitions
- reel-friendly visual storytelling

These additions should enhance the atmosphere without overwhelming the primary subject matter.

## Profile Configuration Note

This addendum does not define MidJourney profiles. Use the project profile YAML file for named profile expansion, including `gimpysticks` and `Chenier-Gothic`, and preserve the exact profile ID order defined there.
