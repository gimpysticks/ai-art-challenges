## File Naming Conventions

Most challenge files follow one of these patterns:

- `@account_#hashtag_MonYY.md` — tied to a specific account and challenge hashtag
- `AccountName_MonYY.md` — named by account/event and date
- `ChallengeTitle_Prompt_List.md` — standalone prompt lists
- `ChallengeTitle_Archive.md` — multi-prompt or session archives
