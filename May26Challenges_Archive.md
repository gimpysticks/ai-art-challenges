# Global Prompt Rules

## MidJourney Rules
- Use MidJourney text prompts only unless explicitly requesting image generation.
- Remove all references to “8k”.
- Keep prompts concise and MJ-friendly.
- Avoid repeated worlds, locations, and motifs.
- Frequently excluded:
  - Lanterns
  - Silent Meridian
  - Ashen Court
  - Clockveil Observatory
  - Cathedral engines
  - Ancient mechanical relics
  - Endless rain
  - Flooded machine canals
  - Repetitive observatories
- Favor originality and non-repetition.

## Output Structure
1. Date / Prompt String
2. MidJourney Prompt
3. Audio Recommendations (multiple tracks preferred)
4. Username(s)
5. Challenge Name / Hashtag
6. Hashtags (maximum five)
7. Story Caption

## Audio Guidance
- Prefer 3–5 recommendations.
- Include title and artist.
- Favor tracks available in Instagram audio.
